# User Controls and Cues

## User Controls
The app is designed to be operated with gloves on, using physical buttons or voice commands.

### Unified Controls
| Action | On-Screen Control | Physical Control | Voice Command | 2s Cue | Behavior |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Launch** | N/A | N/A | N/A | `Start recording with Auto-Zoom in` | App Just Started (Fades after 5s) |
| **Start Recording** | Tap **Record Button** | Double-Click **Any Vol** | "Start", "Record", "Go" | `Recording` | Starts recording |
| **Stop Recording** | Tap **Stop Button** | Double-Click **Any Vol** | "Stop", "Cut" | `Recording Saved` | Stops recording |
| **Zoom In** | N/A | Hold **Volume Up** | "Zoom In", "Closer" | `Zoom In` | Zooms in continuously (say "Hold" to stop) |
| **Zoom Out** | N/A | Hold **Volume Down** | "Zoom Out", "Wider" | `Zoom Out` | Zooms out continuously (say "Hold" to stop) |
| **Stop Zoom** | N/A | Release Button | "Hold", "Lock", "Freeze" |  | Stops any manual zoom |
| **Auto-Zoom** | N/A | Press **Both Vol** | "Auto", "Track" | `Auto Zoom Enabled` | Resumes auto-zoom |
| **Hi-Res Photo** | N/A | N/A | "Photo", "Shoot" | `Photo` | Takes a photo |
| **Show Debug Overlay** | N/A | N/A | "Debug On" | `Debug On` | Shows debug info |
| **Hide Debug Overlay** | N/A | N/A | "Debug Off" | `Debug Off` | Hides debug info |

## User Cues
Visual cues to guide the user, designed to be minimal and short.

## Visual Style
- **Font**: System, Rounded, Bold (Large visibility).
- **Color**: White with black shadow/outline (high contrast for snow).
- **Animation**: Quick Fade In / Slow Fade Out.
